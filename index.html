Orthotic Device Dynamic Simulation

Used lagrangian mechanics to simulate the movement of a model 6-dof device designed to help rehabilite stroke patients. Simulation was done in mathematica. The math is cooler than the video.

Medical Expert System

Prototype decision support system written in Java with the Jess rule engine. Takes parameters of a person and uses AI techniques to provide clinical recommendations based on ACA guidelines.

See Github

Autonomous Pingpong-Ball Shooting Robot

Autonomous Robot built as a 3-person team my senior year of Undergrad. The robot was designed to automatically seek out infared LED's in the base of an opponent robot, and fire a pingpong ball at it, while avoiding objects. Robot was controlled via arduino Mega and designed in Inventor.
